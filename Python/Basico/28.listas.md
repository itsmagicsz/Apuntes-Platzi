# Metodos para las listas

que es?
sirve para almacenar varias valores en una sola variable 

append = Para agregar un elemento de una lista 
pop = Para eliminar un elemnto de una lista

ejemplo 

objetos=[1, 2, true, "Hola"]

objetos.append(25)

[1, 2, true, "Hola", 25]

objetos.pop[true]
